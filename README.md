 ## ⚒️ Como rodar a aplicação | How to run the application
<details>
  <summary> PT </summary> <br>

  Faça Download:
  ```
  git clone git@github.com:queite/job-insights.git
  ```
  Entre na pasta rauz:
  ```
  cd job-insights
  ```
  Instale as dependências:
  ```
  python3 -m pip install -r dev-requirements.txt
  ```
  rode o código
  ```
   flask run
  ```

</details>
